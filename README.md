# Bob-The-Builders

Bob-The-Builders is a modern **React + Vite** demo application designed to simulate a startup ecosystem where **Founders** and **Venture Capitalists (VCs)** interact through dashboards, messaging, and project evaluations.

The project focuses on clean UI design, reusable components, mock data, and a scalable frontend architecture suitable for demos and learning purposes.
